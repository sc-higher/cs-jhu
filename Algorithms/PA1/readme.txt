This file contains several folders:

Analysis - contains PDF of written homework document
input - contains several .dat files to be used for program input
output - folder where results are stored
src - folder with Java source code
bin - folder with Java class files

The Java version used was Java SE 1.8. The IDE used was Eclipse. All programs running on Windows 10.

There are two src files - Driver.java and Main.java.
Please use the Driver file. The Main file contains modifications used for testing purposes.

Usage is as follows:

javac Driver.java
java Driver ../input/<filename> <output-filename>

Input files contain one integer per line with no blank/empty lines. See one of the provided input files for an example.
