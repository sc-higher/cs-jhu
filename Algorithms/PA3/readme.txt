This file contains several folders:

Analysis - contains PDF of written homework document and several tables/graphs.
input - contains several .txt files to be used for program input, as well as a batch file to automate running and required .class files.
output - folder where results are stored
src - folder with Java source code
bin - folder with Java class files

The Java version used was Java SE 1.8. The IDE used was IntelliJ IDEA. All programs running on Windows 10.

There are several source files. The file Driver.java can be used to run the program.

Usage is as follows:

javac *.java
java Driver ../input/<filename> <output-filename>

Input files contain one integer per line with no blank/empty lines. See one of the provided input files for an example.
