The source code in this assignment can be compiled with JDK 8 from the command line. It was developed using IntelliJ IDEA.

To run the program, enter:  java MatrixDeterminant [inputFile] [outputfile].

Input shall be a txt file of the form:

[square matrix 1 size]
[square matrix 1]
[square matrix 2 size]
[square matrix 2]
...
[square matrix n size]
[square matrix n]

e.g. 

3
1 2 3
4 5 6
7 8 9
2
1 0
0 5
